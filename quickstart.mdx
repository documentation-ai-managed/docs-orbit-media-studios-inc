## Prerequisites

<Callout kind="info">
Before you begin, ensure you have:
- A modern web browser like Chrome, Firefox, or Safari
- An email address for account verification
- Optional: A team domain if managing multiple projects
</Callout>

Orbit Media Studios helps you organize project documentation efficiently. Complete the setup in under five minutes.

## Create Your Account

Follow these steps to register.

<Steps>
  <Step title="Visit the Dashboard" icon="monitor">
    Navigate to [https://app.orbitmedia.com](https://app.orbitmedia.com).
  </Step>
  <Step title="Sign Up" icon="user-plus">
    Click **Sign Up** and enter your email, name, and password.
  </Step>
  <Step title="Verify Email" icon="mail">
    Check your inbox for a verification link from Orbit Media Studios and click it.
  </Step>
</Steps>

## Log In and Initial Workspace Setup

Once verified, log in to configure your workspace.

<Tabs>
  <Tab title="Web Browser" icon="globe">
    Enter your credentials at [https://app.orbitmedia.com/login](https://app.orbitmedia.com/login).
  </Tab>
  <Tab title="CLI (Advanced)" icon="terminal">
    Install the CLI and authenticate:

    <CodeGroup tabs="npm,yarn">
```bash
npm install -g @orbitmedia/cli
orbit login
```
```bash
yarn global add @orbitmedia/cli
orbit login
```
    </CodeGroup>
  </Tab>
</Tabs>

After login, set your workspace name and default language in the settings panel.

## Create Your First Documentation Project

Build your first space quickly.

<Steps>
  <Step title="New Project" icon="plus">
    Click **New Documentation Space** in the sidebar.
  </Step>
  <Step title="Configure Basics" icon="settings">
    Enter a name like "Project Alpha Docs", select visibility (`public` or `private`), and add a description.
  </Step>
  <Step title="Add Content" icon="file-text">
    Create your first page with the editor. Use markdown or the visual builder.
  </Step>
</Steps>

<Callout kind="tip">
Save frequently. Your changes auto-publish to the live space.
</Callout>
