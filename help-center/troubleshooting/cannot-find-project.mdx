---
title: Cannot Find My Project
description: Troubleshoot missing project issues in Orbit Media Studios, Inc.
---

## Symptoms

- Project missing from dashboard
- Cannot open project docs

## Solutions

<Steps>
  <Step title="Check Filters">
    Make sure project filters (status or owner) are not hiding your projects.
  </Step>
  <Step title="Refresh Dashboard">
    Click the refresh button or reload the page to update project list.
  </Step>
  <Step title="Contact Support">
    If the project is still missing, email support@orbitmedia.com with project details.
  </Step>
</Steps>
