## Recent Releases

Stay informed about every update to Orbit Media Studios. You access the most recent changes here, organized by version. Each release includes new features, improvements, and bug fixes to enhance your media project management experience.

<Update label="2024-10-15" description="v1.2.0" tags={["feature", "improvement"]}>

## New Features

- Added real-time collaboration for media projects, allowing multiple users to edit assets simultaneously.
- Introduced webhook support for project events like asset uploads and approvals.

## Improvements

- Enhanced media upload performance by `>30%` for large files.
- Updated dashboard UI with customizable widgets for better workflow visibility.

## Bug Fixes

- Fixed asset preview rendering issues in Safari browsers.
- Resolved synchronization errors during offline mode reconnects.

</Update>

<Update label="2024-09-25" description="v1.1.0" tags={["feature", "bugfix"]}>

## New Features

- Integrated AI-powered asset tagging to automatically categorize media files.
- Added export options for projects in ZIP and JSON formats.

## Bug Fixes

- Corrected permission errors when sharing projects with external teams.
- Patched API rate limiting inconsistencies affecting bulk operations.

</Update>

<Update label="2024-08-10" description="v1.0.0" tags={["breaking", "feature"]}>

## Initial Release

- Launched core media project management with asset storage, version control, and team collaboration.
- Implemented secure API endpoints for programmatic access.

## Breaking Changes

- Initial schema updates require re-authentication for existing integrations.

</Update>
